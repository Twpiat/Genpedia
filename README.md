# Genpedia
Web companion app for Genesis LP MUD

Graduation project for the Python Back-End Development Course
Quest database for the Genesis LP MUD

Features:
User Login
Add Quests
Search Quest by Name
List Quests
Show Quest Info
Basic interface
Gamification
